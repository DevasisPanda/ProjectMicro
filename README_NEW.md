# ProjectMicro

## Description
A simple Node.js project for Miravi Mart. This project demonstrates a basic product store and server setup.

## Features
- Product store management
- RESTful API server
- Simple frontend with HTML, CSS, and JavaScript

## Getting Started

### Prerequisites
- Node.js (v14 or above)

### Installation
```bash
git clone https://github.com/DevasisPanda/ProjectMicro.git
cd "New project"
npm install
```

### Running the Application
```bash
node src/server.js
```
The server will start on the configured port (see `src/config.js`).

## Project Structure

- `src/` - Source code (server, config, product store)
- `public/` - Static files (HTML, CSS, JS)
